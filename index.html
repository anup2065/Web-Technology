  1. Write a program to find the factorial of any given number using Javascript.

    Answer:

    The factorial of a number is the product of all positive integers less than or equal to that number.

    JavaScript Code:

    
function factorial(n) {
  let result = 1;
  for (let i = 1; i <= n; i++) {
    result *= i;
  }
  return result;
}

// Example usage
let number = 5;
console.log("The factorial of " + number + " is: " + factorial(number));

    Output: The factorial of 5 is: 120

    Explanation: The loop multiplies numbers from 1 to the given number (5), so 1 × 2 × 3 × 4 × 5 = 120.

    Explain the different operators used in PHP.

    Answer:

    PHP supports various types of operators. They are:

    • Arithmetic Operators: +, -, *, /, % (used for mathematical operations)
    • Assignment Operators: =, +=, -=, *=, etc. (used to assign values)
    • Comparison Operators: ==, !=, >, <, >=, <= (used to compare values)
    • Logical Operators: && (AND), || (OR), ! (NOT)
    • Increment/Decrement Operators: ++, --
    • String Operators: . (concatenation), .= (append)

    What is Object-based programming? Explain different JavaScript objects.

    Answer:

    Object-based programming is a programming model based on the concept of "objects", which can contain data and functions. Unlike object-oriented programming, object-based languages do not support features like inheritance.

    Common JavaScript Objects:

    • Array: Used to store multiple values. Example: let arr = [1, 2, 3];
    • Date: Used to handle dates and times. Example: let today = new Date();
    • Math: Provides mathematical functions and constants. Example: Math.sqrt(25);
    • String: Represents text data. Example: let str = "Hello";
    • Object: General-purpose object to store key-value pairs. Example: let person = {name: "Ram", age: 18};

  36. What is event handling in JavaScript?

    37. Answer:

    Event handling in JavaScript is the process of executing code when specific events occur in a webpage, such as clicks, keypresses, or page loads. This is done using event listeners or event attributes like onclick.

    Example:

    <button onclick="alert('Button clicked!')">Click Me</button>
