  1. Write a program to find the factorial of any given number using Javascript.

    Answer:

    The factorial of a number is the product of all positive integers less than or equal to that number.

    JavaScript Code:

    
function factorial(n) {
let result = 1;
for (let i = 1; i <= n; i++) {
result *= i;
}
return result;
}

// Example usage
let number = 5;
console.log("The factorial of " + number + " is: " + factorial(number));

    Output: The factorial of 5 is: 120

    Explanation: The loop multiplies numbers from 1 to the given number (5), so 1 × 2 × 3 × 4 × 5 = 120.

    1. Connect to the database using PDO.
    2. Query the database to retrieve the specific record(s) you want to display.

$password = '';

} catch (PDOException $e) {
die("Connection failed: " . $e->getMessage());

?>

    2. Query Database for Record

$id = $_GET['id'] ?? 1;

$stmt->execute([$id]);
$user = $stmt->fetch(PDO::FETCH_ASSOC);

if (!$user) {
die("User not found");
}
?>

  3. Write short notes on Internet technology.

    Answer:

    Internet technology refers to the tools and infrastructure used to access, share, and exchange data over the internet. It includes protocols (like TCP/IP, HTTP), hardware (routers, servers), software (browsers, apps), and services (email, cloud storage, VoIP).

    Protocols

    Protocols are rules that define how data is transmitted and received. The most important among these are TCP/IP, which ensures data is properly broken into packets and reassembled at the destination, and HTTP/HTTPS, which manage how websites are loaded, with HTTPS adding a layer of security through encryption. FTP is another protocol used for transferring files across networks.

    Hardware

    Hardware refers to physical components that form the infrastructure of the internet. This includes servers that host websites and data, routers and modems that direct and manage internet traffic, and the physical infrastructure like fiber-optic cables and satellites that carry signals worldwide. Without this hardware, digital communication would not be possible.

    Software

    Software are the programs that allow users to access and use the internet. Software like web browsers like Chrome and Firefox are used to access websites, while email clients help in sending and receiving messages. Various apps allow users to perform specific tasks such as video calling, chatting, or online transactions, making the internet more user-friendly and functional.

    Services

    Services provided through internet technology include email for instant communication, cloud storage that lets users store and access data remotely, and VoIP (Voice over Internet Protocol) which enables voice calls over the internet. These services have replaced many traditional communication and storage methods.

  5. Explain different data types used in JavaScript.

    Answer:

    JavaScript supports various data types, mainly categorized into primitive and non-primitive (reference) types.

    Data Types in JavaScript

    1. Primitive Data Types

    • String: Represents textual data.
      Example: "Hello", 'World'
    • Number: Represents both integers and floating-point values.
      Example: 42, 3.14
    • Boolean: Logical type with values true or false.
      Example: true, false
    • Undefined: Assigned to variables that are declared but not initialized.
      Example: let x; (x is undefined)
    • Null: Represents intentional absence of any object value.
      Example: let user = null;
    • BigInt: Used for integers of arbitrary length.
      Example: 123456789012345678901234567890n
    • Symbol: Unique and immutable primitive, often used as object property keys.
      Example: let sym = Symbol("id");

    2. Non-Primitive (Reference) Data Types

    • Object: Stores collections of key–value pairs.
      Example: { name: "Alice", age: 25 }
    • Array: Special type of object for ordered collections.
      Example: [10, 20, 30]
    • Function: Callable object that encapsulates code.
      Example: function greet() { return "Hello"; }

  9. Explain the different operators used in PHP.

    Answer:

    PHP provides a wide variety of operators to perform different types of operations on variables and values. Here are the main types:

    • Arithmetic Operators: Used for basic math operations like addition (+), subtraction (-), multiplication (*), division (/), and modulus (%).
    • Assignment Operators: Used to assign values to variables. Example: =, +=, -=, *=, etc.
    • Comparison Operators: Used to compare two values. Example: == (equal), != (not equal), >, <, >=, <=
    • Logical Operators: Used to combine multiple conditions. Example: && (AND), || (OR), ! (NOT)
    • String Operators: The dot operator (.) is used to concatenate two strings. Example: "Hello" . " World"
    • Increment/Decrement Operators: ++ and -- are used to increase or decrease a variable by one.

    These operators help perform different logic and calculations in a PHP program.

  11. What is Object-based programming? Explain different JavaScript objects.

    Answer:

    Object-based programming (OBP) is a programming style that uses objects to represent data and behavior but does not support features like inheritance and polymorphism which are found in object-oriented programming (OOP). JavaScript is an object-based language because it allows the creation and use of objects but has limited inheritance support.

    Common JavaScript Objects:

    • Array Object: Used to store multiple values in a single variable. Example: let arr = [1, 2, 3];
    • Date Object: Allows working with dates and times. Example: let today = new Date();
    • Math Object: Provides mathematical constants and methods. Example: Math.sqrt(25);
    • String Object: Used to manipulate text. Example: let str = "Hello";
    • Generic Object: Created using {} to store key-value pairs. Example: let student = {name: "Sita", age: 17};

    These built-in objects make JavaScript powerful and versatile for client-side programming.

    function reverseNumber(number) {
let reversedNumber = 0;
let isNegative = number < 0;
number = Math.abs(number); // Work with absolute value

while (number > 0) {
const digit = number % 10; // Get the last digit
reversedNumber = reversedNumber * 10 + digit; // Append to reversed number
number = Math.floor(number / 10); // Remove the last digit
}

return isNegative ? -reversedNumber : reversedNumber; // Apply sign back
}

Reverse of 0 is: 0

    Explanation: This function reverseNumber takes an integer. It iteratively extracts the last digit using the modulo operator (), builds the reversed number by multiplying the current reversed number by 10 and adding the digit, and then removes the last digit from the original number using Math.floor(number / 10). It also handles negative numbers by reversing the absolute value and then reapplying the negative sign.

  32. Write a JavaScript program to calculate the factorial of a number using the concept of recursion.

    Answer:

    function factorial(n) {
// Base case: if n is 0 or 1, factorial is 1
if (n === 0 || n === 1) {
return 1;
}
// Recursive case: n * factorial of (n-1)
return n * factorial(n - 1);
}

// Example Usage:
console.log(Factorial of 5 is: ${factorial(5)}); // Expected: 120
console.log(Factorial of 0 is: ${factorial(0)}); // Expected: 1
console.log(Factorial of 7 is: ${factorial(7)}); // Expected: 5040
console.log(Factorial of 1 is: ${factorial(1)}); // Expected: 1
// Note: Factorial is typically defined for non-negative integers.
// Handling negative numbers would require additional logic.

    Sample Output:

    Factorial of 5 is: 120
Factorial of 0 is: 1
Factorial of 7 is: 5040
Factorial of 1 is: 1

    Explanation: This JavaScript function factorial calculates the factorial of a given non-negative integer using recursion. The base case is when n is 0 or 1, in which case it returns 1. For any other positive integer n, it recursively calls itself with n−1 and multiplies the result by n. This continues until the base case is reached, unwinding the calls to compute the final factorial.
